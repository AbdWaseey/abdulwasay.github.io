A fully responsive front-end website designed for AURA Sound, a fictional premium headphones brand.
Built using HTML & CSS, this project focuses on clean layout, modern UI, and mobile-first design.

--> Features

Fully responsive design (mobile, tablet, and desktop)

Landing page with hero section

Product highlights and feature sections

Smooth layout using Flexbox/Grid

Modern typography & visual theme

-->Purpose of the Project

This project was created to:

Practice front-end development

Improve responsive layout skills

Showcase UI/UX design ability

Build portfolio-ready projects
